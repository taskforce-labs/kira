# Kira: Review work item is ready to start

Check if the work item is ready to be worked on by assessing the following criteria

- Is the problem articulated concisely, clearly and consistently?
- Is the solution articulated concisely,clearly and consistently?
- Is the work broken down into slices as per `kira slice` conventions

If there are any gaps please ask clarifying questions, put all clarifying questions in a **## Questions** section in the work item. When presenting clarifying questions with selectable options, follow the **kira-clarifying-questions-format** skill (read that skill for the full structure: ## Questions, ### N. [Title], #### Options, checkboxes, Suggested).
