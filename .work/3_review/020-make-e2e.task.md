# make e2e

run the end to end tests from make

## Details

Currently to test commands I run the following:
```bash
bash kira_e2e_tests.sh
```
This is a pain and I want to be able to run the tests via the Makefile.

I want to be able to run the tests via the Makefile like this:
```bash
make e2e
```

People should know about this command and how to use it.
